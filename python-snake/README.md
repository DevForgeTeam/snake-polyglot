# Snake Implementation

A classic Snake game implemented in Python using the `pygame` library. This project is part of the **Snake Polyglot** showcase, which demonstrates how to build the same game in various programming languages following a shared set of rules.

## Dependencies

- Python 3.x
- `pygame`

## Setup and Installation

1. Ensure you have Python 3 installed on your system.
2. Open your terminal or command prompt and navigate to the `python-snake` directory.
3. Install the required `pygame` library using pip:

   ```bash
   pip install pygame
   ```

## Running the Game

To start the game, run the following command from the `python-snake` directory:

   ```bash
   python main.py
   ```

## Game Controls

- **Move Up:** `W` or `Up Arrow`
- **Move Down:** `S` or `Down Arrow`
- **Move Left:** `A` or `Left Arrow`
- **Move Right:** `D` or `Right Arrow`
- **Restart:** `SPACE` (Available when Game Over)
